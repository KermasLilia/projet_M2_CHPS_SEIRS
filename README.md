# Projet de Synthèse : Modélisation Épidémiologique SEIRS

**Auteur :** KERMAS Lilia
**Master 2 CHPS**

Ce dépôt contient le code source du projet comparant deux approches de modélisation épidémiologique :
1.  **Partie 1 (ODE)** : Résolution numérique d'équations différentielles (C et Python).
2.  **Partie 2 (SMA)** : Simulation Multi-Agents stochastique (C, C++ et Python).

##  Contenu
* `Partie1/` : Codes Euler/RK4 et notebook d'analyse.
* `Partie2/` : Codes multi-agents et notebook d'analyse statistique.
* `Rapport_projet_synthèse_Kermas_Lilia_.pdf` : Le rapport complet.

##  Installation
1.  Installer les dépendances Python : `pip install -r requirements.txt`
2.  Compiler les codes C/C++ : `make` dans chaque dossier.
